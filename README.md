## 🕹️ Game Features (Mobile Optimized)

### 1. Flick Mode (Precision)
The core experience for mobile. Test your tap-to-target speed. Targets appear at random locations and disappear quickly.
* **Difficulties:** Easy, Medium, and Hardcore (each with unique visual vibration intensities).
* **UI/UX:** Buttons and targets scaled for optimal touch interaction.
* **Game Over:** Reaching 10 misses ends the session.

## 🎨 Creative Profile

This project showcases a blend of technical development and artistic vision:
* **Concept Art:** Original Cyberpunk "Glitch" aesthetic, focusing on high-contrast Magenta and Cyan themes.
* **Web Design:** Mobile-first responsive layout using Flexbox and CSS Transforms for perfect centering on any screen.
* **Front-End Dev:** Pure Vanilla JavaScript logic for game state and target spawning.

## 🛠️ Tech Stack

* **HTML5:** Semantic structure.
* **CSS3:** Advanced animations using `@keyframes`, pseudo-elements (`::before`, `::after`), and **Media Queries** for mobile scaling.
* **JavaScript (ES6+):** DOM manipulation, touch event optimization, and interval management.

## ✨ Key Features

* **Dynamic Vibrations:** Visual feedback that triggers on interaction, maintaining the "unstable technology" vibe.
* **Fast Response:** Optimized for low latency on mobile browsers.
* **Glitch UI:** A sleek, immersive aesthetic developed from my background as a Concept Artist.
